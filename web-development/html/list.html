List Lesson

ကိုယ်ဖော်ပြချင်တဲ့ အကြောင်းအရာတစ်ခုကို list အနေနဲ့ ဖော်ပြချင်တဲ့အခါမှာ အသုံးပြုလို့ရနိုင်မဲ့ List နဲ့ ဆိုင်တဲ့ HTML Element (၃) မျိုးရှိပါတယ်။

  1. <ul> = Unorderd List → <ul> ထဲမှာ ထည့်ထားတဲ့ List တွေကို Bullet နဲ့ဖော်ပြပါတယ်။
  2. <ol> = Orderd List → <ol> ထဲမှာ ထည့်ထားတဲ့ List တွေကို နံပါတ်စဉ်နဲ့ ဖော်ပြပါတယ်။
  3. <li> = List Item → <li> ဆိုတာ List အနေနဲ့ ဖော်ပြချင်တဲ့ အကြောင်းအရာပါ။

<ul> = Unorderd List
